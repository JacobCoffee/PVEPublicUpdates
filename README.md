# PVEPublicUpdates
Setup Public Updates for Proxmox VE

## How To Run

 - Call
   - `https://github.com/JacobCoffee/PVEPublicUpdates/blob/a43e0d42b44309242eca03a6ee662af17c5ddc9d/updates.sh`
**OR**
   - `https://raw.githubusercontent.com/jacobcoffee/pvepublicupdates/master/updates.sh`
 - Run `./updates.sh`

## Permission Issues
- Run `chmod +X updates.sh`
